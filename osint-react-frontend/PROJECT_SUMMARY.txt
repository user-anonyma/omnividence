INSTALLATION & QUICK START
================================================================================

PREREQUISITES:

1. cd osint-react-frontend
2. npm install
3. cp .env.example .env
4. npm start

BUILD FOR PRODUCTION:
npm run build

================================================================================
API INTEGRATION
================================================================================

The frontend connects to Flask backend endpoints:

POST   /api/upload              # Upload single or multiple images
GET    /api/search              # Search by image ID
POST   /api/search/batch        # Batch search multiple images
GET    /api/results/{id}        # Get search results
GET    /api/sources             # Get available sources
GET    /api/health              # Health check

See INTEGRATION_GUIDE.md for detailed API documentation and requirements.
